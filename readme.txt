Плагин экспортирует товары в OZON маркетплейс. Для работы требуется установленный Woocommerce плагин.

== Description ==
Плагин экспортирует товары в OZON маркетплейс.
Для начала работы необходимо ввести OZON Client-Id и OZON Api-ключ на странице плагина (выберите 'Ozon Export' в меню административной части)
Для добавления товара в набор для экспорта отметьте его в блоке 'Ozon export' на странице редактирования товара. Введите необходимые для ozon параметры: высота, ширина, глубина и вес товара.

Экспортируется название товара, описание, цена, атрибуты, главное изображение и все картинки.
Товары экспортируются на Озон раз в сутки.
Плагин умеет работать как с простыми товарами, так и с вариативными. Вы можете экспортировать часть описания, как обязательный параметр для OZON (например, состав)

Плагин экспортирует товары в те категории, которые уже добавлены. Для добавления новой категории Вам необходимо обратиться к разработчику на email ozepdevelopment@mail.ru 
Если планируется экспортировать товары с атрибутами, необходимо будет сообщить ярлыки атрибутов и ярлыки их значений. Это необходимо для корректной привязки атрибута товара к категориям OZON.
Плагин спроектирован таким образом, что относительно просто позволяет дорабатывать механику под задачи и автоматизацию конкретного магазина/каталога.
По вопросам помощи в установке, настройки и доработки: ozepdevelopment@mail.ru
Будем рады сотрудничеству.

== Frequently Asked Questions ==

= Как устроен механизм экспорта? =

Для экспорта используется Ozon Seller API — программный интерфейс для работы с маркетплейсом Ozon. Он даёт возможность обмениваться информацией между системой продавца и Ozon.
Запрос со списком товаров отправляется в формате Json.

= Как получить доступ к Seller API =

1. Зарегистрируйтесь в Ozon Seller. Подробнее о регистрации в Базе знаний продавца.
2. Получите API-ключ.

= Как получить API-ключ = 

1. В личном кабинете в Ozon Seller перейдите в Настройки → API ключи.
2. Придумайте название для ключа и нажмите Создать ключ — появится список ваших API-ключей.
